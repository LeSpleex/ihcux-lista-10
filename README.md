## Explicação Das Telas
Essas telas mostram o fluxo de um aplicativo mobile para bares/restaurantes, com foco em pedido digital e gestão de fila. A ideia principal é facilitar a experiência do cliente — desde entrar no app até avaliar o estabelecimento.

#### TELA 1: Login / Cadastro
Primeiras telas: usuário entra com username e senha ou cria uma conta.
Objetivo: identificar o cliente para pedidos e histórico.

#### TELA 2: Lista de estabelecimentos
O usuário vê opções como:
Bar do Zezé
Bar do Mou
Também há abas tipo “Bares”, “Restaurantes”, “Cinema”.
Objetivo: escolher onde quer consumir.

#### TELA 3: Página do estabelecimento
Mostra o bar selecionado (ex: Bar do Zezé – aberto).
Exibe categorias de produtos:
Cervejas
Cachaças
Vodkas
Vinhos
Porções
Não alcoólicos
Objetivo: navegar no cardápio.

#### TELA 4: Seleção de produtos
Lista de itens com:
Nome (ex: Corona, Heineken)
Quantidade
Preço
O usuário monta o pedido.
Botão: “Entrar na fila”

#### TELA 5: Sistema de fila
Mostra a posição do cliente:
Ex: “Você está em 3 de 8”
Pode sair da fila.
Objetivo: organizar atendimento sem aglomeração ou confusão

#### TELA 6: Avaliação
Após a experiência, o usuário avalia o bar com estrelas
Objetivo: feedback e reputação do local.
